SDA Commons Client Jersey Example¶

This example module shows an application that uses the client-jersey bundle to invoke another service.

Beside the initialization of the bundle, it includes the generation of three different clients: * Generic Platform client * Generic External client * External client using an interface that describes the API to generate a proxy * Generic External client with custom configurations

The integration test shows how to set up a mock using the sda-commons-client-jersey-wiremock-testing module.

Check the documentation for more details.
